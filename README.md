## Run Locally

Clone the project

```bash
  git clone https://github.com/AldaErwin/Ecommerce-Erwin-3
```

Go to the project directory

```bash
  cd Ecommerce-Erwin-3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
